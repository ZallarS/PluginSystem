# Моя MVC система с плагинами

## Установка

1. Скопируйте файлы на сервер
2. Настройте конфигурацию в `app/config/`
3. Установите зависимости: `composer install`
4. Настройте веб-сервер на папку `public/`

## Структура

- `app/` - Исходный код приложения
- `public/` - Публичные файлы
- `storage/` - Файлы системы
- `vendor/` - Зависимости Composer

## Создание плагина

1. Создайте папку в `app/plugins/`
2. Создайте главный файл плагина
3. Реализуйте `PluginInterface`
4. Активируйте плагин через админку или конфиг
